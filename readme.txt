

PROJET SD_GC

Ce projet contient les fichiers suivants :
- projet.c
- projet.h
- main.c
- cvrp_100_1_c.dat
- cvrp_100_1_r.dat
- cvrp_100_1_det.dat
- exemple.dat

Un Makefile est fourni pour faciliter la compilation et l'exécution du projet.

méthode d'utilisation du Makefile :

1-compiler le projet : commande : make
2-exécuter le projet : commande : make run FILE=nom_fichier.dat

exemple d'utilisation:

make
make run FILE=exemple.dat
